# My App
A mobile app built with Expo and React Native.

## Features

- Bottom tab navigation
- Home weather display
- Weather data based on current or nearest city
- Australian news list
- User profile page
- About page navigation
- Camera screen example
- Image carousel

## Run the Project
- install dependencies:
```bash
npm install
```

- Start the app:
```bash
npx expo start -c
```

Open iOS Simulator:

```text
Press i
```

## API Setup

Add your NewsAPI key in `utils/api.js`:

```js
const NEWS_API_KEY = 'your NewsAPI key'
```

Weather data is provided by the WeatherAU API.
